# Greenden 🌿

Greenden is a responsive front-end website built using HTML and Tailwind CSS.  
It is a modern plant shop / nursery themed project created to practice Tailwind utility classes and responsive web design.

## Features
- Responsive design for mobile and desktop
- Modern landing page UI
- Product showcase section
- Clean navigation bar
- Tailwind CSS styling
- Beginner-friendly project structure

## Technologies Used
- HTML5
- Tailwind CSS

## Purpose
This project was built to improve my skills in:
- Tailwind CSS
- Responsive layouts
- Flexbox and Grid
- UI design practice
